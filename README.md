# Employee Directory

## API Used

This project uses the Random User Generator API.

API Endpoint

https://randomuser.me/api/?results=20

The API provides randomly generated user profiles including:

- Name
- Email
- Phone
- Country
- Profile Picture

## Technologies

- HTML
- CSS
- JavaScript
- Fetch API

## Features

- Fetches live user data
- Loading indicator
- Error handling
- Search users by name
- Refresh data
- Responsive design
